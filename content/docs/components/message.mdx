Composable primitives for rendering **user** and **assistant** turns in a chat UI: alignment, bubble chrome, **[Streamdown](https://github.com/vercel/streamdown)**-powered markdown via **`MessageMarkdown`**, optional **`MessageAvatar`** (shadcn **Avatar** wrappers), **`MessageActions`** for icon buttons, and **[Attachments](/docs/components/attachments)** for file chips above the bubble.

## Usage

```tsx keepBackground
import {
  Message,
  MessageStack,
  MessageContent,
  MessageMarkdown,
  MessageActions,
  MessageActionGroup,
  MessageAction,
  MessageAvatar,
  MessageAvatarImage,
  MessageAvatarFallback,
} from "@/components/nexus-ui/message";
```

```tsx keepBackground
<Message from="user">
  <MessageStack>
    <MessageContent>
      <MessageMarkdown>Hello</MessageMarkdown>
    </MessageContent>
  </MessageStack>
</Message>
```

## Examples

### With Actions

Use **`MessageActions`** and **`MessageActionGroup`** for a row of controls. **`MessageAction`** is a thin **Slot** for individual action buttons.

<DemoWithCode src="components/nexus-ui/examples/message/with-actions.tsx">
  <MessageWithActions />
</DemoWithCode>

### With avatar

**`MessageAvatar`**, **`MessageAvatarImage`**, and **`MessageAvatarFallback`** wrap the shadcn **Avatar** primitives. Place the avatar **after** **`MessageStack`** for **`from="user"`**, and **before** **`MessageStack`** for **`from="assistant"`**.

<DemoWithCode src="components/nexus-ui/examples/message/with-avatar.tsx">
  <MessageWithAvatar />
</DemoWithCode>

### With attachments

Render **`AttachmentList`** / **`Attachment`** inside **`MessageStack`** above **`MessageContent`**. Use **`readOnly`** on **`Attachment`** when showing files that were already sent (no remove control or progress). See **[Attachments](/docs/components/attachments)** for **`AttachmentMeta`** and variants.

<DemoWithCode src="components/nexus-ui/examples/message/with-attachments.tsx">
  <MessageWithAttachments />
</DemoWithCode>

### Rich text (markdown)

**`MessageMarkdown`** renders markdown with **Streamdown** and shared prose-style classes. Pass a string as **`children`** (or a template literal) for headings, lists, code blocks, and links.

  <MessageRichText />
</DemoWithCode>

## API Reference

### Message

Root row: **`justify-end`** for **`from="user"`**, **`justify-start`** for **`from="assistant"`**. Wraps children in a **`max-w-[90%]`** flex row. Provides context for nested primitives.

### MessageStack

Vertical **`gap-2`** column; **`items-end`** for user, **`items-start`** for assistant (from context).

Bubble wrapper: gray background and padding for **user**; transparent for **assistant**. **`from`** overrides context when needed.

### MessageActionGroup

Horizontal **`gap-1`** group for action buttons.

### MessageAction

Defaults to a **`div`**; with **`asChild`**, merges props onto the child (**Radix Slot**). Use with **`Button`** **`variant="ghost"`** **`size="icon-sm"`** for icon-only controls.

### MessageAvatar   

`MessageAvatar`, `MessageAvatarImage`, and `MessageAvatarFallback` accept the same props as their shadcn Avatar counterparts. See the [shadcn Avatar documentation](https://ui.shadcn.com/docs/components/avatar) for more details.
