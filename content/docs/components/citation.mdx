Inline **source** pills on a **Hover Card**: favicon, label, and a preview with title, description, and link row. Pass **`citations`** as **`{ url, title?, description? }`**; URLs drive **favicons** and a short **site name** from the hostname.

**`Citation`** provides list context and (with **`CitationCarousel`**) the active slide. Item-scoped primitives (**`CitationTitle`**, **`CitationDescription`**, …) read **`CitationCarouselItem`** when present, otherwise the carousel’s **active** citation.

## Usage

```tsx keepBackground
import {
  Citation,
  CitationCarousel,
  CitationCarouselContent,
  CitationCarouselHeader,
  CitationCarouselIndex,
  CitationCarouselItem,
  CitationCarouselNext,
  CitationCarouselPagination,
  CitationCarouselPrev,
  CitationContent,
  CitationDescription,
  CitationFavicon,
  CitationFaviconGroup,
  CitationItem,
  CitationSiteName,
  CitationSource,
  CitationTitle,
  CitationTrigger,
} from "@/components/nexus-ui/citation";
```

Single source: **`CitationTrigger`** + **`CitationContent`** + **`CitationItem`** + field primitives. Multiple sources: **`CitationCarousel`** with **`CitationCarouselItem`** **`index`**, header row, and pagination controls.

```tsx keepBackground
<Citation citations={[{ url: "https://example.com", title: "Example", description: "…" }]}>
  <CitationTrigger />
  <CitationContent>
    <CitationItem>
      <CitationTitle />
      <CitationDescription />
      <CitationSource />
    </CitationItem>
  </CitationContent>
</Citation>
```

## Examples

### Multiple sources

With more than one **`citations`** entry, the default chip shows **`+N`** after the first source. The card body uses **`CitationCarousel`** so **`CitationCarouselPrev`**, **`CitationCarouselNext`**, and **`CitationCarouselIndex`** stay in sync with Embla.

<DemoWithCode src="components/nexus-ui/examples/citation/multiple-sources.tsx">
  <CitationMultipleSources />
</DemoWithCode>

### Trigger label and favicon

Default chip variations: **favicon only** (**`showSiteName={false}`**), **numeric id** (**`showFavicon={false}`** **`label="[14]"`**), **hostname** (**`label`** from **`parseCitationUrl(url).hostname`**), and **resolved site name** (**`label`** from **`resolveCitationSource(...).siteName`**).

<DemoWithCode src="components/nexus-ui/examples/citation/trigger-variants.tsx">
  <CitationTriggerVariants />
</DemoWithCode>

### Inline with text

Three short **paragraphs** of body copy with chips **inside** the sentences. Wrap each **`Citation`** in **`inline-block align-middle`** (and a light **`translate-y`**) so the pill aligns with the line. Because the hover-card **root** is block-level, model each paragraph as a **`div`** (or another block wrapper)—not a **`<p>`**—so the markup stays valid when **`Citation`** sits mid-sentence.

### Types and helpers

Exported for typing and pipelines:

- **`CitationProps`**, **`CitationTriggerProps`**, **`CitationContentProps`** — root and slot prop types
- **`CitationSourceInput`** — `{ url: string; title?: React.ReactNode; description?: React.ReactNode }`
- **`ResolvedCitation`** — normalized **`url`**, **`title`**, **`description`**, **`siteName`**, **`faviconSrc`**
- **`parseCitationUrl(urlStr)`** — tolerant URL parse for display and favicon host
- **`resolveCitationSource` / `resolveCitationSources`** — map inputs to **`ResolvedCitation`**
- **`rootDomainSiteName(url)`** — short label from hostname (heuristic, not a full PSL)
