## Usage

```tsx keepBackground
import { toast, Toaster } from "@/components/nexus-ui/toaster";
```

```tsx keepBackground
<Toaster />
```

```tsx keepBackground
toast.default("Draft saved");
toast.success("Saved");
toast.info("New update available");
toast.warning("Connection is unstable");
toast.error("Request failed");
toast.loading("Syncing...");
```

```tsx keepBackground
toast.success("Profile updated", {
  description: "Your account details have been saved.",
  action: {
    label: "Undo",
    onClick: () => console.log("Undo clicked"),
  },
});
```

## API Reference

### Toaster

Container component that mounts Sonner. This wrapper sets `theme` from `next-themes` by default, enables class hook `toaster group`, and defaults `toastOptions` to `{ unstyled: true }` (overrideable via props).
